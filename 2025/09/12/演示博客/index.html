示例文章：Hexo 归档与标签演示

2025-09-12 | 技术

本文为示例文章，用于演示 Hexo 的归档（archives）和标签（tags）功能。

写在前面

  • 在文章 front-matter 中使用 tags 和 categories 字段可以自动生成对应的标签页与分类页。
  • 主题须包含相应的模板（多数主题已内置），且站点根目录 _config.yml 需启用生成器（hexo-generator-tag / hexo-generator-archive / hexo-generator-category）。

示例 front-matter（已在本文件顶部展示）

在本文章顶部的 YAML front-matter 中已添加：

  • tags: hexo, 教程
  • categories: 技术

生成与预览（在项目根目录，Windows PowerShell / CMD）：

浏览器打开 http://localhost:4000 查看归档与标签页：

  • 归档页（如已配置）通常在 /archives/ 或 /archives/index.html
  • 标签页通常在 /tags/ 下按标签列出

小结

添加 tags 与 categories 非常简单：在每篇文章的 front-matter 中声明即可。修改完配置后运行 hexo clean && hexo g && hexo s 验证效果。
